# KIN Non-Technical Onboarding Script

## Purpose

Give a concierge or operator a simple sequence for onboarding a non-technical KIN user.

## Script

### 1. Explain what KIN is
"KIN is your personal AI companion. It’s friendly and easy to talk to, but it’s also powerful behind the scenes. You’ll use it mainly through chat, starting with Telegram."

### 2. Explain the supported path
"Right now, the primary supported setup is Telegram. We’ll connect you there first because it’s the simplest and safest way to get started."

### 3. Pair the user
- Confirm the user has Telegram installed.
- Send or display the pairing instructions.
- Have the user send the pairing code in a direct message.
- Confirm pairing succeeded before describing any advanced features.

### 4. Set expectations
"KIN can answer questions, help with creative and technical tasks, and eventually use tools when allowed. It will not take high-risk actions without your approval."

### 5. Introduce privacy and safety plainly
"It won’t ask you to paste secret keys into chat, and it won’t control anything sensitive unless you clearly approve it."

### 6. Give the first-use prompt
Suggested first prompt:
"Hi KIN, tell me what you can help me with and how you work."

### 7. For advanced users
If the user asks about local/offline or specialized creative-web capabilities, explain that those are part of the planned extended KIN stack and may not be active in the first baseline release.

## Success Condition

The user reaches the Telegram chat, understands the approval model, and can start with one safe first interaction without technical setup knowledge.
