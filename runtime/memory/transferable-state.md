# Transferable Companion State

## Goal

Define what follows Cipher when ownership changes.

## Transferable State Includes

- learned skills
- improved workflows
- non-personal customizations
- accumulated competence
- character-consistent product tuning

## Transferable State Does Not Include

- private personal memories
- secrets
- owner-specific sensitive history
- hidden residue from the previous relationship

## Product Rule

The NFT transfers the evolving companion artifact, not the previous owner's private self.
