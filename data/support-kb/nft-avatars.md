# NFT Avatars and GLB Identity

Kin companions use GLB (GL Transmission Format Binary) 3D avatars linked to Solana NFTs for on-chain identity verification.

## Understanding NFT Avatars

### What is a GLB Avatar?
- 3D model file in GLB format
- Animated character representation of your Kin
- Stored on decentralized storage (IPFS/Arweave)
- Linked to Solana NFT for ownership verification

## Viewing Your Avatar

### In Mission Control
- Open Mission Control
- Your Kin's card shows the animated GLB avatar
- Interact by hovering/clicking for animations

### In External Viewers
- GLB files can be viewed in any 3D viewer
- Download from the NFT metadata
- Compatible with VR/AR applications

## NFT Verification

### On-Chain Identity
- Each Kin has a unique Solana NFT
- NFT proves ownership and authenticity
- Metadata includes GLB URL and traits

### Verifying Ownership
1. Check the NFT in your Solana wallet
2. Verify mint address matches Kin record
3. Confirm GLB URL in metadata

## Troubleshooting

### Avatar Not Loading
- Check internet connection
- Clear browser cache
- Verify GLB URL is accessible

### NFT Not Showing
- Confirm wallet connection
- Check Solana network status
- Verify NFT is in correct wallet

### Animation Issues
- Update browser to latest version
- Enable WebGL in browser settings
- Try a different browser

## Technical Details

- GLB format for efficient 3D delivery
- Draco compression for smaller files
- Skeletal animation support
- PBR materials for realistic rendering
